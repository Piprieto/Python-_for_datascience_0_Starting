This repository contains the exercises for **Training Piscine Python for Data Science – Module 0: Starting**.

The goal of this module is to provide a **basic introduction to Python**, focusing on first contact with the language and its core concepts, using **Python 3.10**.
It is designed for beginners or anyone who wants to refresh their fundamentals before moving on to more advanced topics in data science.

---

##  Objectives

* Get familiar with **Python 3.10**
* Understand basic Python syntax
* Learn how to run Python scripts
* Practice fundamental programming concepts
* Build a solid foundation for future data science modules

---

##  Topics Covered

* Python environment setup
* Basic syntax and indentation
* Variables and data types
* Simple functions
* Control flow (if, loops)
* Basic input and output
* First Python scripts


---


##  Notes

This module is intended as a **learning exercise**, not a production project.
The focus is on clarity, understanding, and correct use of Python basics rather than optimization.
